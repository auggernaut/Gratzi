About

Gratzi are tokens of gratitude.

Gratzi are also:

  • not copied to any centralized service (distributed)
  • sent only to the person you designate (private)
  • encrypted (confidential)
  • signed (authentic)
  • generated by open-source client-side javascript (open)
  • stored in your personal storage provider (yours)

How it works

You send a Grat to your friend and he replies with a Zi. This bi-directional value transfer.

So what?

We believe that gratitude, if expressed consistently and meaningfully, can not only make us feel good but also be tremendously useful.

Our hope is that by creating enough Gratzi we can:

  1. see how flows value directly, without need of a currency
  2. identify who to help and how in order to reach a common goal.

We believe that an open standard for creating and sharing Gratzi's could help usher in a new wave of people-centric technologies and services. We are a part of the unhosted movement, where applications and data are separate.

And it's also much more than that...

Mission:

To make a universal system of value exchange without middlemen.

To create the worlds first viral unhosted web app.

Distributed

This web application is running on <%= url %>, but it can run anywhere. Even on your local machine. Download, modify, and run it however you like.

Personal

Every Gratzi you create is stored in your DropBox, Google Drive, GitHub, or wherever you have cloud storage. They don't go anywhere else. A Gratzi is between you and the person you send it to.

Open

This is just the starting point. Anyone can change how Gratzi works or create new tools for it. Since no single organization owns this (see distributed and personal) it's free to evolve organically.

The source code for this application is available in this repository on GitHub.
